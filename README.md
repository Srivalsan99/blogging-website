# Blogging Website

A full-featured blogging platform 

## Features
- User registration and login with secure password hashing
- Create, edit, and delete personal articles with WYSIWYG editor
- Comment system with nested replies (up to 2 levels)
- Like articles and view total likes
- User profile management (username, bio, avatar, etc.)
- Responsive and clean user interface
- Proper input validation and access control

## Technologies
- Java + Spring Boot
- MySQL Database
- HTML, CSS, JavaScript
- Thymeleaf / Handlebars (templating)

## Setup
1. Clone the repository
2. Configure database settings in `.env` or `application.properties`
3. Run the provided `db-init.sql` script
4. Start the application
